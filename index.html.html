How to Use SnapGrade

1. Sections: Go to the Sections tab to create a class. Click the section to manually add students or bulk import them via CSV.

2. Exams: Go to Exams to create an answer key. Set the number of items and tap the bubbles to set the correct answers. You can also print the OMR sheet from here.

3. Assign: On the Home tab, tap "Assign" to link an Exam to a Section.

4. Scan: Tap "Scan Papers" on your assignment. Align the 4 square markers on the printed OMR sheet within the camera brackets. The app will auto-detect and grade.

5. Review & Export: Click the assignment on the Home tab to see results, edit them, and export detailed item analysis.
